Atualização Necessária

Esse erro acontece porque precisamos criar as "gavetas" novas no banco de dados para guardar o Roteiro, Visual e Hora.

1 Copie o Código SQL abaixo: 

-- 1. Tabela Social Posts (Novas Colunas)
ALTER TABLE public.social_posts 
ADD COLUMN IF NOT EXISTS plataformas TEXT,
ADD COLUMN IF NOT EXISTS descricao_visual TEXT,
ADD COLUMN IF NOT EXISTS conteudo_roteiro TEXT,
ADD COLUMN IF NOT EXISTS hora_agendada TIME DEFAULT '10:00';

-- 2. Tabela Clientes (Link de Contexto)
ALTER TABLE public.clientes
ADD COLUMN IF NOT EXISTS link_conteudos_anteriores TEXT;

2 Execute no Supabase:

  1. Acesse o painel do seu projeto no Supabase (SQL Editor).
  2. Cole o código copiado na área de edição.
  3. Clique no botão Run (ou Executar).
  4. Importante: Recarregue a página do sistema após rodar!
